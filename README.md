# HexSoftwares_Iris_Data_Exploration
Iris dataset exploration using Python with data visualization and analysis.

 Project Overview
This project is part of my Data Science Internship at Hex Softwares.

The goal of this task is to explore the Iris dataset and understand its structure using different data visualization techniques.


 Steps Performed
- Loaded the Iris dataset using sklearn
- Converted dataset into a pandas DataFrame
- Checked for missing values
- Explored dataset using summary statistics
- Visualized data using:
  - Histograms
  - Pairplots
  - Boxplots
  - Heatmaps
  - Countplots

 Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

 Key Observations
- No missing values found in the dataset
- Setosa species is clearly distinguishable
- Petal length and width are strongly correlated


Conclusion
This project helped in understanding how to perform basic data exploration and visualization, which is an important step in any data science project.
